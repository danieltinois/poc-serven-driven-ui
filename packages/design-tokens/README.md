# @poc/design-tokens

Tokens compartilhados entre o CMS Angular e o app Flutter.

Arquivos:

- `tokens.json`: fonte legivel para humanos.
- `tokens.css`: variaveis CSS usadas no Angular.
- `tokens.dart`: constantes usadas no Flutter.

Nesta POC os arquivos sao mantidos manualmente para facilitar a explicacao. Em producao, uma ferramenta como Style Dictionary poderia gerar `tokens.css` e `tokens.dart` a partir de uma fonte unica.
